Readme :

- These visualisations were implemented by using HTML, JavaScript, and D3.js. To view the visualisations, please run the command "python -m http.server" in the terminal and open the url 'localhost:8000' on the browser. 

(If the page cannot be run on the localhost, you can view these visualisations on the site: https://pasitk.github.io/visualisation-uk-student-sex/ instead.)

- All datasets and Jupyter Notebook files for data analysis are in the folder 'dataset-and-analysis'.

- All datasets' file names have been renamed from the original names for the convenience to use in analysis and visualisation. The original sources of all datasets are indicated in each Jupiter Notebook file.
